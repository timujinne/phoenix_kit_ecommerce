2 @moduledoc """
3 Context for managing product options.
4
5 Provides a two-level option system:
6 - **Global options** - Apply to all products (stored in shop_config)
7 - **Category options** - Apply to products in specific category (stored in category.option_schema)
8
9 When retrieving options for a product, the system merges global and category options,
10 with category options overriding global ones by key.
11
12 ## Localization Note
13
14 Option labels and values are currently stored as plain strings, not localized JSONB maps.
15 This means options display the same in all languages. Future enhancement: convert
16 option schema to support localized labels like `"label" => %{"en" => "Material", "ru" => "Материал"}`.
17
18 ## Usage
19
20 # Get/set global options
21 Options.get_global_options()
22 Options.update_global_options([%{"key" => "material", "label" => "Material", "type" => "text"}])
23
24 # Get/set category options
25 Options.get_category_options(category)
26 Options.update_category_options(category, [%{"key" => "mounting_type", ...}])
27
28 # Get merged schema for a product
29 Options.get_option_schema_for_product(product)
30
31 # Validate product metadata against schema
32 Options.validate_metadata(product.metadata, schema)
33
34 ## Price Calculation
35
36 Options with `affects_price: true` can modify the final product price.
37 Two modifier types are supported:
38
39 - `fixed` - Add exact amount (e.g., +$10)
40 - `percent` - Add percentage of base price (e.g., +20%)
41
42 ## Allow Override
43
44 Options with `allow_override: true` can have their price modifiers customized
45 per-product. Override values are stored in product metadata under `_price_modifiers`.
46 When calculating price, the system checks for overrides first, then falls back
47 to the default values from the option schema.
48
49 Calculation order:
50 1. Sum all fixed modifiers (checking for overrides)
51 2. Add to base price (intermediate price)
52 3. Sum all percent modifiers (checking for overrides)
53 4. Apply percent to intermediate price
54
55 Example:
56 - Base price: $20
57 - Material: PETG (+$10 fixed)
58 - Finish: Premium (+20% percent)
59 - Final: ($20 + $10) * 1.20 = $36
